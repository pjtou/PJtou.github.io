WinVega 7

27 Dec 2016, Montreal

Introduction

Le nouveau WinVega comprend des grosses nouveautés surtout sur le module Indien.  Le logiciel a peu changé pour les méthodes occidentales.

Il fonctionne comme l'ancienne version.  Ce logiciel est très riche et peut prendre du temps pour comprendre toutes les options.  Prenez le temps de vous familiariser avec le fonctionnement, cela ne sera sans doute pas instantanné.

La doc n'est pas prète dans cette version Beta pour les options nouvelles, elle viendra en ligne d'ici quelques temps sur mon site, alors voici quelques indices :

Les cartes tropicales ou sidérales se font  de la meme facon qu'avant.  Avec le bouton Calculate this Chart.  Pour le bouton Surya Siddantha Graha voir par. 3.

La page des Options est très riche et peut etre consultée pour comprendre les possibilités.

Le module Indien a de nombreux boutons par clic-droit sur les controles disponibles, un peu partout.

Par ex sur les cartes (diamand ou en carré) un petit bouton au centre ouvre un sous menu par clic droit.   Toutes les divisions (Vargas) de Parasara sont disponibles.

Sur les listes de périodes un clic droit donne accès a de très nombreuses options.  Ce sous menu ouvre par ex. la panoplie des périodes Indiennes, Grecques etc.   Ces périodes et leur configuration sont la spécialité de Winvega.

N'hésitez pas a faire des clic droits un peu partout.  Il n'y a aucun danger pour la santé ;)

Les Vargas sont des thèmes spécialisés.  Le navamsa par ex. est le thème du mariage, de la prédilection spirituelle et dharmique d'une personne.   Comme pour les Sages de l'Inde la vie matérielle est moins omniprésente que dans notre civilisation, le navamsa est le thème le plus important.

Le dasamsa montre l'impact professionnel, les hauts et les bas.

Le saptamsa la thème de la dynastie, des enfants, de relations, rencontres etc

Voir les indications pour chaque varga.

Paragraf 3:  (facultatif)

Le bouton Surya Siddantha Graha ne fonctionne que si un thème a été sauvegardé auparavant grace au logiciel Kundalee.  Il faut faire une petite maneuvre avec l'Explorer de Windows pour renommer et replacer un petit fichier dans le dossier de Winvega, voici les détails :

Kundalee calcule des cartes selon la Surya Siddantha.  Ce procédé est unique, il n'existe aucun autre outil pour cette technique originale a l'astrologie ancienne.  Ces méthodes sacrées ne sont données qu'a de rares personnes dans ce monde ci. 

il s'agit du calcul des vraies grahas (planètes en tant que divinités) qui se pratiquait probablement aussi en occident dans l'antiquité.  Ptolémée dans son Amalgeste décrit les 7 Sphères célestes qui sont la meme mécanique astrale du Surya Siddantha et des autres traités Siddantha de l'Inde.  Les scientifiques se sont heurtés a ces méthodes sans vraiment chercher a les comprendre, puisque l'astronomie n'est pas intéréssée par ce qui sort du domaine des sens physiques.   C'est pourquoi la mécanique physique astronomique, aussi décrite par le Surya S. a perdu de son coté, les formules physiques des planètes visibles, inutiles de son point de vue.  Seul l'astrologie ayant une portée dans nos vie, selon eux.

Il faut d'abord telecharger Kundalee, et apprendre a le manipuler, dresser des thèmes etc. Lorsque un thème est calculé et montré a l'écran, Kundalee place sur le disque (dans son dossier) un petit fichier : TRUEPLANETS.TXT.  Ceci contient les positions des planètes en tant que grahas.

(notez que nous avons deja placé dans le dossier de Winvega, plusieurs thèmes connus, (dont le mien) qui peuvent servir a vous familiariser avec ces grahas.  Ces thèmes SS deja prèts n'ont donc pas besoin de Kundalee.  Soyez sur d'avoir les coordonnées deja entrée dans les cases de dates, heures et latitudes, longitudes, avant de cliquer sur Surya Siddantha. Ceci pour le calcul des Maisons.)

Mais avec Kundalee, ce fichier Trueplanets.txt peut etre exploité par Winvega.  Commencer par renommer ce fichier ainsi :   TRUEPlanetsJean.TXT  ou bien TruePlanetsPaul.txt   

(garder les mots Trueplanets avec un nom, avec l'extention TXT)

Puis placer ce fichier dans le dossier de Winvega 7.   Ensuite ce fichier peut etre reconnu par Winvega.

N'oubliez pas que ce ficher Trueplanets.txt est temporaire et effacé dès lors qu'une autre carte est calculé avec Kundalee, ou si Kundalee est arrété.

Dans Winvega, il faudra ensuite rentrer les memes coordonnées de la meme personne.  La maneuvre complète est un un peu longue au début, mais après quelques manipulations, devient rapide.

Une fois ces coordonnées prètes dans Winvega, cliquez sur le bouton Surya Siddanta Graha.  Allez chercher le fichier TrueplanetsPaul.txt que vous aviez placé dans ce dossier de Winvega auparavant.   Une fois trouvée, faite un double clic dessus.  Cette carte est montrée l 'écran, identique a Kundalee, mais selon une présentation un peu plus aisée.

En bas a droite de la roue apparait le mot SuryaS qui indique des positions de planètes selon le mode Surya Siddantha.  Pour avoir la meme carte du ciel, choisissez les maisons Sripati.   Cette méthode est disponible aussi dans le module Indou avec tous les vargas, amitiés, Shad Balas, Dasas, périodes, Aspects etc
